GimpDatafileData

typedef struct {
  const gchar *filename;
  const gchar *dirname;
  const gchar *basename;

  time_t       atime;
  time_t       mtime;
  time_t       ctime;
} GimpDatafileData;

This structure is passed to the GimpDatafileLoaderFunc given to gimp_datafiles_read_directories() for each file encountered in the data path.
